# 🏠 RentFinder

A full-stack rental marketplace platform that connects busy individuals with rent finders.

---

## 🚀 Features

- ✅ Request creation (busy users)
- ✅ Task system (rent finders)
- ✅ Escrow payment flow
- ✅ Manual payment verification
- ✅ Admin dashboard
- ✅ Analytics dashboard

---

## 🧑‍💼 Roles

- **Busy Individual** → creates requests
- **Rent Finder** → accepts and completes tasks
- **Admin** → verifies payments and manages the platform

---

## 🛠 Tech Stack

Frontend:
- React + TypeScript
- Vite
- React Query
- Recharts

Backend:
- Node.js + Express
- Supabase (PostgreSQL)
